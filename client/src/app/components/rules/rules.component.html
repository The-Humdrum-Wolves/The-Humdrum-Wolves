Age Limitations

Every user should be 18+ years old.

Game rules

Bet the final score of a fixture

Win points as follows:

Correct Suggestion - 3 points

Incorrect Suggestion - 0 points

Ranking

The game ranking is produced by the users total points won on different fixtures betting. The user with highest result takes the 1st place in the ranking.

Prizes

Only the first three players in ranking win prizes.

1st place - Tickets for a match in English Premier League

2nd place - An original football jersey (Chosen by the user)

3rd place - An original La Liga ball
